## 六、根据答复调整做法（内部参考）

| 若平台说… | 我们则… |
|-----------|---------|
| 必须嵌平台 | 重点做 **Agent JSON + 原型**，前端可能平台做 |
| 先独立站演示 | **网页 + 部署** 主力推进 |
| 暂时不给 Agent 清单 | 先做 **模板 + 自造样板 Agent**，清单到了再迁 |
| Skill 列表未就绪 | skill_chain 先 **写约定 id**，Skills 组就绪后对齐 |
| 登记即可、不必跑通 | 专注 **agent.yaml + 展示页**，执行 demo 由 Skills 组配合 |
